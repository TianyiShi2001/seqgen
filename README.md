# seqgen

DNA/RNA/Protein Sequence Generator

# Examples

## Basics

```bash
seqgen 1000 3            # 3 random 1000-nucleotide DNA seqs
seqgen -p 1000 3         # 3 random 1000-aa protein seqs
```
